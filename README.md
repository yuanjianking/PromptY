# Prompt 优化工作台 (Prompt Optimization Workbench)

## 📖 项目简介

Prompt 优化工作台是一个系统化的提示词(Prompt)工程工具，帮助用户分析、评估和优化 Prompt。通过多维度分析和多策略生成，提升与 AI 模型的交互效果。

## ✨ 核心功能

### 1. **用户输入层**
- 使用场景描述（原始文本、场景类型、使用目的等）
- 原始 Prompt 输入
- 目标模型配置（支持 DeepSeek、OpenAI 等）
- 约束条件设置（格式、长度、风格等）
- 全局生成参数调节

### 2. **信息充分性评估**
- Prompt 结构识别（目标段、指令段、输出段、隐含假设）
- 完整度等级评估
- 模型能力画像（推理能力、创造性、格式遵循、幻觉风险）
- 信息缺失清单
- 生成决策建议

### 3. **Prompt 事前分析**
- 场景拆解（显性任务、隐性子任务、边界、非目标）
- 目标显式化（主目标、次级目标、优先级、冲突检测）
- 不确定性标注（含糊词语、未说明要求、模型依赖点）
- 风险预测（指令忽略、输出偏移、幻觉等）

### 4. **多策略 Prompt 生成**
- **精确型**：信息准确、结构清晰
- **探索型**：创意发挥、开放自由
- **强约束型**：严格遵循框架
- **分析型**：逻辑推理、逐步推导
- **角色导向型**：特定角色视角

### 5. **指标与问题说明**
- 核心指标评估
- 信息缺失点识别
- 模型敏感点分析
- 偏差类型预警

### 6. **事后解释器**
- 偏差归因分析
- 改进方向建议
- Prompt 对比解释


## 🔧 配置说明

### API 配置

在页面右上角点击"⚙️ 配置API"按钮，可以临时设置：
- **API密钥**（必填）
- **API地址**（默认：https://api.deepseek.com）
- **模型名称**（默认：deepseek-chat）

> 注意：配置仅在当前页面会话中有效，刷新页面后需重新配置。

### 环境变量（可选）

创建 `.env` 文件可以设置默认配置：

```env
VITE_API_KEY=your-api-key
VITE_API_BASE_URL=https://api.deepseek.com
VITE_MODEL_NAME=deepseek-chat
```

## 🎯 使用流程

1. **填写输入信息**：在用户输入层填写场景描述、原始Prompt、目标模型和约束条件
2. **配置全局参数**：调整探索度、风险偏好等参数
3. **提交分析**：点击"提交优化会话"按钮
4. **查看分析结果**：各区域会逐步显示分析结果
5. **优化迭代**：根据分析结果调整输入，再次提交
